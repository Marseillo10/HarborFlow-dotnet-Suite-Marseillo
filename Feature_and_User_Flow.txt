HarborFlow Application - Web & WPF

This document outlines the user flow for both the WPF and Web applications.

[Web App Status: In Development - Core features are being refined. Guest access is limited.]

│
├─── 🚀 Aplikasi Dimulai
│    └─── 🏠 Jendela Utama (MainWindow / Web App) Ditampilkan
│         │
│         ├─── 👤 Mode Pengguna: Tamu (Default)
│         │    │
│         │    ├─── 🗺️ Navigasi -> Peta Kapal (MapView)
│         │    │    ├─── Fitur: Lacak kapal secara real-time dengan ikon khusus. [Backend Terverifikasi] [Web Implementasi]
│         │    │    └─── Fitur: Ganti lapisan peta (Street, Satellite, Nautical). [Web Implementasi]
│         │    │
│         │    ├─── 📰 Navigasi -> Berita Maritim (NewsView)
│         │    │    ├─── Fitur: Lihat berita dari berbagai kategori (All, International, National, Official). [Web Implementasi]
│         │    │    ├─── Fitur: Filter berita dengan kata kunci. [Web Implementasi]
│         │    │    └─── Fitur: Filter berita berdasarkan rentang waktu (24h, 7d, 30d, All) dengan auto-scroll. [Web Implementasi]
│         │    │
│         │    ├─── 🚢 Navigasi -> Manajemen Kapal (VesselManagementView)
│         │    │    └─── Fitur: Lihat daftar semua kapal di database. [Web Implementasi]
│         │    │
│         │    ├─── 📋 Navigasi -> Permintaan Layanan (ServiceRequestView)
│         │    │    └─── Fitur: Lihat semua permintaan layanan (hanya baca). [Web Implementasi]
│         │    │
│         │    ├─── 🔐 Header -> Tombol "Login"
│         │    │    └─── Aksi: Membuka halaman Login. [Web Implementasi]
│         │    │
│         │    └─── ➕ Header -> Tombol "Register"
│         │         └─── Aksi: Membuka halaman Registrasi. [Web Implementasi]
│
└─── 🔑 Pengguna Melakukan Login
     │
     └─── 👤 Mode Pengguna: Terotentikasi
          │
          ├─── 📊 Navigasi -> Dasbor (DashboardView)
          │    ├─── Fitur: Lihat grafik status permintaan layanan. [Backend Terverifikasi] [Web Implementasi]
          │    └─── Fitur: Lihat grafik jumlah kapal per tipe. [Backend Terverifikasi] [Web Implementasi]
          │
          ├─── 🗺️ Navigasi -> Peta Kapal (MapView)
          │    ├─── Fitur: Lacak kapal secara real-time dengan ikon khusus. [Backend Terverifikasi] [Web Implementasi]
          │    └─── ⭐ Fitur Login: Panel Bookmark
          │         ├─── Aksi: Tambah bookmark dari tampilan peta saat ini. [Web Implementasi]
          │         ├─── Aksi: Pilih bookmark untuk navigasi cepat. [Web Implementasi]
          │         └─── Aksi: Hapus bookmark yang tersimpan. [Web Implementasi]
          │
          ├─── 📰 Navigasi -> Berita Maritim (NewsView)
          │    └─── (Fungsionalitas sama seperti mode Tamu)
          │
          ├─── 🚢 Navigasi -> Manajemen Kapal (VesselManagementView)
          │    │    (Akses penuh untuk Admin)
          │    ├─── Fitur: Tambah, Edit, Hapus data kapal. [Backend Terverifikasi] [Web Implementasi]
          │    └─── Fitur: Lihat daftar semua kapal di database. [Backend Terverifikasi] [Web Implementasi]
          │
          ├─── 📋 Navigasi -> Permintaan Layanan (ServiceRequestView)
          │    │    (Akses penuh untuk Petugas & Admin)
          │    ├─── Fitur: Lihat semua permintaan layanan. [Backend Terverifikasi] [Web Implementasi]
          │    └─── Fitur: Setujui atau Tolak permintaan. [Backend Terverifikasi] [Web Implementasi]
          │
          └─── 🚪 Header -> Tombol "Logout"
               └─── Aksi: Kembali ke Mode Tamu. [Web Implementasi]
